# MICA GD Pack

An extension pack to simplify setting up VS Code for GD 220 and GD 431.

## Included Extensions:
  1. [**Auto Rename Tag** by Jun Han](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)
  1. [**Beautify** by HookyQR](https://marketplace.visualstudio.com/items?itemName=HookyQR.beautify)
  1. [**Bracket Pair Colorizer** by CoenraadS](https://marketplace.visualstudio.com/items?itemName=CoenraadS.bracket-pair-colorizer)
  1. [**CSS Minify** by DiegoMolina](https://marketplace.visualstudio.com/items?itemName=DiegoMolina.css-minify)
  1. [**CSS Peek** by Pranay Prakash](https://marketplace.visualstudio.com/items?itemName=pranaygp.vscode-css-peek)
  1. [**Intellisense for CSS class names in HTML** by Zignd](https://marketplace.visualstudio.com/items?itemName=Zignd.html-css-class-completion)
  1. [**Live Server** by Ritwick Dey](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
  1. [**Path Intellisense** by Christian Kohler](https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense)
